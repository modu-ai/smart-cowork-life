# 슬기로운 직장생활

> 직장인·취준생을 위한 한국형 업무 스킬 플러그인

Claude Cowork에서 바로 사용할 수 있는 **12가지 업무 스킬**과 **5가지 슬래시 커맨드**를 제공합니다.

## 설치 방법

1. `smart-office-life.plugin` 파일을 다운로드합니다
2. Claude 데스크톱 앱의 Cowork 모드에서 파일을 열면 자동으로 설치됩니다
3. 또는 `.claude/plugins/` 폴더에 직접 복사합니다

## 포함 스킬 (12종)

| 스킬 | 설명 | 트리거 키워드 |
|------|------|---------------|
| **비즈니스 이메일** | 상황별 이메일 작성 (요청/회신/사과/공지) | 이메일, 메일, email |
| **회의록** | 회의록 자동 생성 및 액션아이템 추출 | 회의록, 회의, 미팅 |
| **한국형 비즈니스 문서** | 기획안, 보고서, 품의서 등 문서 작성 | 보고서, 기획안, 품의서 |
| **제안서/견적서** | 프로젝트/사업 제안서 및 견적서 작성 | 제안서, 견적서, proposal |
| **엑셀 업무 자동화** | 매출 집계, 급여 계산, KPI 대시보드, 동적 배열 함수 | 엑셀, Excel, 스프레드시트 |
| **프롬프트 엔지니어링** | AI 활용 프롬프트 설계 5원칙 + 26기법 | 프롬프트, prompt, AI 활용 |
| **프로젝트 관리** | WBS, 간트차트, 이슈 트래커, RACI | 프로젝트, WBS, 일정 관리 |
| **이력서·포트폴리오** | 이력서/자소서/포트폴리오/링크드인 | 이력서, 자기소개서, 포트폴리오 |
| **데이터 보고서** | 데이터 분석 및 시각화 보고서 | 데이터, 분석, 차트, 대시보드 |
| **한영 번역** | 비즈니스 문서 한영/영한 번역 (7개 업종) | 번역, translate, 영작 |
| **PPT 디자인 시스템** | Pretendard+명조 타이포그래피, 6:3:1 색상, 한국형 레이아웃 | PPT 디자인, 슬라이드, 발표자료 |
| **SVG 도식화** | Mermaid v11 + 커스텀 SVG로 비즈니스 다이어그램 생성 | 도식화, 다이어그램, SVG, 플로우차트 |

## 슬래시 커맨드 (5종)

| 커맨드 | 설명 |
|--------|------|
| `/daily-report` | 일일업무보고서 빠른 작성 |
| `/weekly-report` | 주간업무보고서 빠른 작성 |
| `/quick-email` | 상황 설명만으로 이메일 즉시 생성 |
| `/meeting-note` | 회의 내용을 회의록으로 빠른 정리 |
| `/resume-check` | 이력서·자기소개서 검토 및 피드백 |

## 내장 폰트

| 폰트 | Weight | 용도 |
|------|--------|------|
| **Pretendard** | 9종 (Thin~Black) | 제목, 본문, KPI 숫자, 장식 |
| **조선일보명조** | Regular | 인용구, 격식 문구, 포인트 강조 |

## 사용 예시

```
"거래처에 납품일정 변경 요청 이메일 써줘"
→ 비즈니스 이메일 스킬 자동 활성화

"이번 주 회의 내용 정리해줘"
→ 회의록 스킬 자동 활성화

/weekly-report 고객사 미팅 3건 완료, 제안서 초안 작성, 다음 주 최종 PT 예정
→ 주간업무보고서 자동 생성

/meeting-note 참석: 김과장, 이대리. 3월 프로젝트 착수 합의.
→ 회의록 자동 생성
```

## v1.2.0 업데이트 내역

- 신규 커맨드: `/weekly-report`, `/meeting-note` 추가
